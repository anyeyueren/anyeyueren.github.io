list介绍

​ 链表是一种物理存储单元上非连续、非顺序的存储结构，数据元素的逻辑顺序是通过链表中的指针链接次序实现的。链表由一系列结点组成。可以高效的进行插入和删除，但是不支持随机访问。list是一个双向链表。

特性：

(1) 采用动态存储分配，不会造成内存浪费和溢出；

(2) 链表执行插入和删除操作十分方便，修改指针即可，不需要移动大量元素；

(3) 链表灵活，但是空间和时间额外消耗较大；

元素插入和删除

  1. 在链表头部进行操作：

    (1) 在容器头部插入元素： push_front(elem);

    (2) 在容器头部删除元素： pop_front(elem);

  2. 在尾部插入或者删除元素：

    (1) 在容器尾部插入一个元素： push_back();

    (2) 在容器尾部删除一个元素： pop_back();

  3. 在中间插入元素：

    (1) 在pos处插入元素elem，返回新数据位置： insert(pos,elem);

    (2) 在pos处插入n个elem元素，无返回值： insert(pos,n,elem);

    (3) 在pos位置插入[beg,end)区间的数据，无返回值： insert(pos,beg,end);

  4. 移除容器中所有元素：

​ clear() :删除列表容器中的所有元素：调用它们的析构函数，然后将它们从列表容器中删除，使其size为 0。

  1. 删除元素：

(1) 删除pos处的数据，返回下一个数据的位置： iterator erase ( iterator position );
(2) 删除[first, last)区间的数据，返回下一个数据位置： iterator erase ( iterator first, iterator last );

(3) 从列表中删除具有特定值的所有元素： void remove ( const T& value );

remove_if函数

​ remove_if函数的参数都是迭代器，调用该函数后，会从list中删除谓词op条件返回为true的所有元素，会调用析构函数释放这些元素并减小list本身的size。

谓词 pred 可以实现为任何类型表达式，它采用一个与列表相同类型的参数并返回一个布尔值，使用时会将beg和end之间所有元素传入到op(elem)中，op会返回true或者false，remove_if函数会将搜所有true对应的元素删除。

resize函数

  1. 重新指定容器的长度为num： resize(num); resize(num,elem);

若容器变长，则以默认值(第二个使用elem填充)填充新位置。

若容器变短，则末尾超出容器长度的元素被删除。

字符串排序sort

​ sort成员函数有两种调用方式，如果直接调用sort()，采用默认的从小到大方式排序。 也可以编写函数自定义排序规则进行排序。

链表合并merge

  1. void merge ( list<T,Allocator>& x );

  2. ​ template

​ void merge ( list<T,Allocator>& x, Compare comp );

注： 将x合并到list，将 x 的所有元素插入到列表对象各自的有序位置。 这会清空 x 并增加list大小。第二个版本（模板函数），具有相同的行为，但需要一个特定的函数来执行负责确定插入点的比较操作。

unique函数

​ unique函数用于删除list当中所有重复的元素，只留下第一个，之外的全部清除。当然也可以自己定义“重复的行为”。

链表拼接splic

​ 将选中的元素从列表x移动到列表容器的指定位置，有效地将指定元素插入容器并从x中删除它们。这会通过插入的元素数量增加容器大小，并以相同的数量减少 x 的大小。
