string

​ string是STL的字符串类型，通常用来表示字符串。

c语言中常使用char *表示一个字符串，string和char *的比较：

(1) string是一个类，而char *是一个指向字符的指针；

(2) string不用考虑内存越界和释放；

(3) string提供了很多操作字符串的函数，例如find，copy，erace，replace等；

string构造函数

string(); //构造一个空的字符串string s1。

string(const string &str); //构造一个与str一样的string。如string s1(s2)。

string(const char *s); //用字符串s初始化

string(int n,char c); //用n个字符c初始化

string迭代器

​ string中迭代器有正向迭代器iterator和逆向迭代器reverse_iterator，但是需要注意的是在使用时iterator是与**string::begin()和string::end()一起使用；而reverse_iterator是与string::rbegin()和string::rend()**一起使用的。

string成员函数

string遍历

数组方式 operator[ ] 以“str[i]”的方式获取字符串中的元素
迭代器方式 string::at() 以“str.at(i)”的方式获取字符串中的元素

operator[]和at()二者是有区别的（越界时）：

​ 区别在于at()在越界时会抛出异常，[]在刚好越界时会返回(char)0，再继续越界时，编译器直接出错。如果你的程序希望可以通过try,catch捕获异常，建议采用at()。

string与char*互换

(1) string ——–> char*(c_str)

(2) char * ——-> string

string赋值

  1. 把字符串s赋给当前的字符串：

    string & string::assign(const char *s);

    string &assign(const string &s);

  2. 把字符串s的前n个字符赋给当前的字符串：

*string &assign(const char s, int n);

  1. 用n个字符c赋给当前字符串：

    string &assign(int n,char c);

  2. 把字符串s中从start开始的n个字符赋给当前字符串：

    string &assign(const string &s,int start, int n);

字符串拼接

string查找

  1. 从pos(默认数值是0)开始查找**字符c**在当前字符串的位置：

    string::find(char c, int pos=0);

  2. 从pos开始查找**字符串s**在当前字符串的位置（如果查找不到，就返回-1）：

*string::find(const char s, int pos=0);

string::find(const string &s, int pos=0);

  1. 从pos开始查找字符串s的前n个字符位置：

string::find(const char s, size_t pos, size_t n);*

注：如果逆向查找就是rfind函数。

string替换

  1. 将从pos处开始的n个字符替换成字符串s：

*string & string::replace (int pos, int n, const char s);

string & string::replace (int pos, int n, const string &s);

  1. 用str替换迭代器起始位置和结束位置 的字符：

string& string::replace (const_iterator i1, const_iterator i2, const string& str);

string& string::replace (const_iterator i1, const_iterator i2, const char str);*

  1. 用substr的指定子串（给定起始位置和长度）替换从指定位置上的字符串：

string& string::replace (size_t pos, size_t len, const string& substr, size_t subpos, size_t sublen);

  1. 用s的前n个字符替换从开始位置pos长度为len的字符串：

string& string::replace(size_t pos, size_t len, const char* s, size_t n);

  1. 用s的前n个字符替换指定迭代器位置(从i1到i2)的字符串：

string& string::replace (const_iterator i1, const_iterator i2, const char* s, size_t n);

  1. 用重复n次的c字符替换从指定位置pos长度为len的内容：

string& string::replace (size_t pos, size_t len, size_t n, char c);

  1. 用重复n次的c字符替换从指定迭代器位置（从i1开始到结束）的内容：

string& string::replace (const_iterator i1, const_iterator i2, size_t n, char c);

string插入

(1) 在index位置插入count个字符ch：

​ basic_string& insert( size_type index, size_type count, CharT ch );

(2) index位置插入一个常量字符串/字符串对象：

​ basic_string& insert( size_type index, const CharT s );*

​ basic_string& insert( size_type index, const basic_string& str );

(3) index位置插入常量字符串中的前count个字符：

​ basic_string& insert( size_type index, const CharT s, size_type count );*

(4) index位置插入常量str的从index_str开始的count个字符：

​ basic_string& insert( size_type index, const basic_string& str, size_type index_str, size_type count );

(5) index位置插入常量str从index_str开始的count个字符，count可以表示的最大值为npos.这个函数不构成重载：

​ basic_string& insert( size_type index, const basic_string& str,size_type index_str, size_type count = npos);

注：npos表示一个常数，表示size_t的最大值，string的find函数如果未找到指定字符，返回的就是一个npos。

(6) 在迭代器pos处插入字符ch且返回指向该字符迭代器：

​ iterator insert( iterator pos, CharT ch );
​ iterator insert( const_iterator pos, CharT ch );

(7) 迭代器指向的pos位置插入count个字符ch：

​ void insert( iterator pos, size_type count, CharT ch );

​ iterator insert( const_iterator pos, size_type count, CharT ch );

string比较

​ C++中string的compare()比较函数：

  • 字符串相同，返回0；
  • 调用字符串 < 被调用字符串，返回-1；
  • 调用字符串 > 被调用字符串，返回1；

注：字符串说的大小通常和字典顺序是一致的。另外，compare()比较时逐字符比较的，一旦能比较出结果，就不再继续比较了。

string提取子串

返回由pos开始的n个字符组成的子字符串:

​ string& substr(int pos=0, int n=npos) const;

string长度

容量

size 返回字符串中字符数的计数
length 返回字符串中字符数的计数(同size,别名)
max_size 字符串对象可以具有的最大字符数
resize 调整字符串大小
capacity 返回字符串对象中分配的存储空间的大小
reserve 请求字符串中分配的存储空间的容量至少为多大的数值
clear 被设置为一个空字符串，删除任何以前的内容，从而使其大小为0个字符
empty 测试字符串是否为空

获取元素的方式

operator[] 以“str[i]”的方式获取字符串中的元素
at 以“str.at(i)”的方式获取字符串中的元素

修改字符串

operator+= 在原字符串后追加内容
append 在原字符串后追加内容
push_back 在原字符串后面添加字符
assign 将新内容分配给替换其当前内容的字符串
insert 通过在字符串内容的特定位置插入一些额外的内容来扩展当前字符串内容
erase 擦除部分字符串内容，缩短字符串的长度
replace 用通过参数确定的其他内容替换当前字符串的一部分
copy 从字符串复制字符序列
swap 与另一个字符串交换内容

字符串操作

c_str 生成与字符串对象具有相同内容的以空字符结尾的字符序列，并将其作为指向字符数组的指针返回。
data 返回与字符串内容相同的字符数组的指针,不会附加结尾的空字符
get_allocator 返回用于构造对象的分配器对象
find 在字符串中查找相关内容
rfind 在字符串中寻找最后出现的相关内容
find_first_of 返回字符串中第一个匹配项的位置
find_last_of 返回字符串中最后一个匹配项的位置
find_first_not_of 返回字符串中第一个不符合匹配的位置
find_last_not_of 返回字符串中最后一个不符合匹配的位置
substr 提取字符串中的子串
compare 字符串比较大小
